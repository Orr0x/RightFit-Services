# User Stories

This folder contains user stories for the RightFit Services platform.

## Organization

Stories are organized by sprint and feature area:

- **Sprint Stories**: Stories planned for specific sprint iterations
- **Backlog**: Future stories not yet assigned to sprints
- **Completed**: Archived completed stories for reference

## Story Template

Each story should follow this structure:

```markdown
# [STORY-ID]: Story Title

**Sprint**: Sprint X
**Priority**: High/Medium/Low
**Story Points**: X
**Estimated Time**: X days

## User Story

**As a** [user role]
**I want** [goal/desire]
**So that** [benefit/value]

## Acceptance Criteria

- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

## Technical Notes

[Implementation details, dependencies, etc.]

## Definition of Done

- [ ] Implementation complete
- [ ] Tests written and passing
- [ ] Code reviewed
- [ ] Documentation updated
- [ ] Deployed to staging
```
